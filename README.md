# GrafiCando 🎴

GrafiCando es un juego educativo de cartas diseñado para aprender y practicar conceptos de estadística y análisis de datos de una manera divertida e interactiva.

## Características 📊

- **Tres tipos de cartas:**
  - 🃏 Comodines: Cartas especiales con efectos únicos
  - ❓ Preguntas: Diferentes tipos de preguntas sobre estadística
  - 📚 Datos Curiosos: Información interesante sobre estadística y matemáticas

- **Tipos de preguntas:**
  - Análisis de gráficos
  - Verdadero/Falso
  - Alternativas
  - Preguntas visuales

## Tecnologías Utilizadas 🛠️

- HTML5
- CSS3 (con Tailwind CSS)
- JavaScript (Vanilla)
- Tailwind CSS (a través de CDN)

## Instalación 🚀

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/GrafiCando.git
```

2. Navega al directorio del proyecto:
```bash
cd GrafiCando
```

3. Abre el archivo `index.html` en tu navegador web preferido.

## Uso 🎮

1. Al iniciar el juego, verás tres mazos de cartas diferentes.
2. Haz clic en cualquier mazo para sacar una carta.
3. Según el tipo de carta:
   - Preguntas: Responde la pregunta y verifica tu respuesta
   - Datos Curiosos: Lee la información interesante
   - Comodines: Sigue las instrucciones especiales
